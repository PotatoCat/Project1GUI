
Download Processing Here:
https://processing.org/download/

Open the ImgManipulation Folder
Open ImgManipulation.pde in Processing
Press the Play Button in the top left

Select an image file, use the tools to edit your image as you please
Bug: You cannot Resize a Cropped Version of the Image. Resize will just Resize as if the Image was never cropped.

Save Your Image or select a New File to edit
